LoggerBaseBuildJson - метод
Формирование строки JSON по данным, переданным в журнализатор.

Пространство имён:  PZone.Logging
Сборка:  PZone.Core (в PZone.Core.dll) Версия: 1.1.0.0 (1.1.0.0)
Синтаксис
C#
Копировать
protected static string BuildJson(
	string appId = null,
	string appVersion = null,
	string message = null,
	Object data = null,
	Exception exception = null
)

Параметры

appId (Optional)
Тип: SystemString
Идентификатор приложения.
appVersion (Optional)
Тип: SystemString
Версия приложения.
message (Optional)
Тип: SystemString
Заголовок сообщения.
data (Optional)
Тип: SystemObject
Данные сообщения.
exception (Optional)
Тип: SystemException
Данные исключения.

Возвращаемое значение

Тип: String
Метод возвращает строку объекта JSON в формате
Копировать
{
    AppId: <app-id>,
    AppVersion: <app-version>,
    Message: <message>,
    Data: <data>,
    Exception: <exception>
}
Если какие то свойства объекта не заданы (равны null), то они не попадают в результат.
